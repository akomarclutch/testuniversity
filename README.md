# credouniversity

Enrollment API

## Usage

### Pre-reqs

Requires some es6 features, node 6.x+ recommended (developed on 7.2)

### Setup

```
npm install
```

### Run

test

```
npm test
```

serve (defaults to port 8080)

```
npm start 
```

Swagger docs available at /documentation for GUI based testing
